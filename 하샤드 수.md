# 하샤드 수

### 알고리즘

1. x를 10으로 나누면서 mod 연산을 하여 한자리 씩 추출하여 더한다.
2. x/sum * sum == x 이면 하샤드 수 이다.



### 코드

```c++
#include <string>
#include <vector>

using namespace std;

bool solution(int x) {
    bool answer = true;
    int sum=0, y=x;
    while(y>0){
        sum+=y%10;
        y/=10;
    }
    if(((x/sum) * sum)==x) answer= true;
    else answer= false;
    return answer;
}
```
