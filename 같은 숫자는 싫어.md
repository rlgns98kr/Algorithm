# 같은 숫자는 싫어

### 알고리즘

1. arr[0]을 answer에 push한다.
2. i 1\~arr.length 까지 반복하며 arr[i-1]!=arr[i]일 때(다른 숫자가 최초로 나올 때) arr[i]를 answer에 push한다.



### 코드

```javascript
function solution(arr)
{
    var chk=[];
    var answer = [];
    
    answer.push(arr[0]);
    for(let i=1; i<arr.length; i++){
        if(arr[i-1]!=arr[i]){
            answer.push(arr[i]);
        }
    }
    
    return answer;
}
```
